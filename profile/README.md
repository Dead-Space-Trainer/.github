**Installation Guide**  
1. Download the latest archive.  
2. Extract to a dedicated folder.  
3. Run `DeadSpaceTrainer.exe` as administrator.  
4. Launch Dead Space.  
5. Press INSERT to open the trainer interface.  
6. Activate features via hotkeys or menu.  

Re-attach after game restarts or chapter loads.

**Safety & Risk Information**  
- **High risk of save file corruption**: Extreme resource values, instant progression, or enemy edits can permanently damage your save. Always create multiple backups before every session.  
- Heavy modifications during autosave cycles frequently result in lost progress or broken story flags.  
- Rapid changes may cause temporary scripting glitches or animation issues.  
- The trainer process commonly triggers antivirus detections due to memory injection.  
- Test all features in a new playthrough first. No recovery possible for corrupted saves.
